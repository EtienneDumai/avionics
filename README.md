# But du projet

Ce projet est destiné à reprendre du niveau sur des langages bas niveau (C,C++) ainsi que leurs concepts associés (programation orientée objet, gestion mémoire, programation concurencielle etc ...). Pour ce faire le projet réaliser est un siulateur de commande de vol d'un avion (en simplifiant certain principe).

# Concept à voir

Les concepts à voir sont : 
- POO
- Allocation mémoire dynamique
- Threads
- Mutex

# Déroulé du projet

Durant ce projet le but est d'incrémenter petit à petit les utilisations des concepts à voirs listé plus hauts. Une fois que ces concepts ont été mis en place, dans un deuxième temps le but serait de faire de ce programme un backend via une technologie permettant de faire une API en C++ (WebSocket par exemple) pour ensuite le connecter à un frontend qui affichera des informations en fonction de ce qu'il reçoit.

# Utilisation de l'IA

Durant ce projet je vais utilisé l'IA, mais l'IA ne donnera jamais le code directement, elle va être utiliser comme un tuteur qui va me faire réfléchir pour que je comprenne ce que j'écrit et çela renforce vraiment mes connaissances sur les concepts vu plus haut.
